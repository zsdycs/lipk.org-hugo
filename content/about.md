## 我 { #me }

一个在苏州念书，并就业的中山人。  
村中身高扛把子，双眼度数 500+ ，甚爱大黑框。  
单手可提一袋 30 公斤装的大米，跳起来可以摸到篮筐。  
会爬树 🌳 ，会锄地 🎑 ，会游泳 🏊 ，会做饭 🍚 ，会吹牛 🎤 ，会绘画🎨。  
会在下雨天睡觉，也会在起雾的玻璃上画心。

## 内容 { #content }

1. 博客凌乱，希望标题达意，内容详略，愿君可于字句间有所收获。
2. 本网站的各内容页都有版权标识，若能得欣赏，望转载标注来源。
3. 反对抄袭与简单的复制粘贴，若发现本站有此现象，可严厉批评。
4. 若见内容错漏、超链失效，愿寄言告，若**编辑**提交合并，将感泣。
